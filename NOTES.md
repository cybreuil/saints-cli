library crate for core logic and data structures of saints-cli (saints lookup / data parsing)
This library crate contains the core logic and data structures for the saints-cli application. It handles the retrieval and parsing of saint data, providing a clean API for the command-line interface to interact with.

binary crate for saints-cli command-line interface
This binary crate implements the command-line interface for the saints-cli application. It utilizes the saints_cli library crate to fetch and display the saint of the day or for a specified date.
